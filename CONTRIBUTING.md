# Contributing to Research Agent Chatbot

## Welcome!

Thanks for your interest in contributing!

## Before You Start

1. Read DEVELOPMENT.md for branch strategy
2. Check existing feature branches
3. Understand the architecture

## Development Setup

```bash
# Clone
git clone <repo-url>
cd research-agent-chatbot

# Install dependencies
pip install -r requirements.txt

# Run tests
pytest tests/
```

## Branch Strategy

We use individual feature branches:
- Each branch = one focused feature
- Clean, reviewable commits
- See docs/development/branch_strategy.md

## Making Changes

1. Create feature branch
2. Make focused commits
3. Push and create PR

## Code Quality

- Write tests for new features
- Follow existing code style
- Update documentation
- Keep commits focused

## Commit Messages

Use semantic commit messages:
- feat: New feature
- fix: Bug fix
- docs: Documentation
- test: Tests
- refactor: Code refactoring

## Questions?

Open an issue or discussion!
